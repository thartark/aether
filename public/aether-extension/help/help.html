Aether Help Guide

Everything you need to know about using Aether - your AI-powered answer engine.

Getting Started

  • Add an API key: Click the extension icon, then use the onboarding panel or go to Settings to add a key. Groq is free and fast!
  • Select text: Highlight any text on a webpage. The floating toolbar appears above your selection.
  • Click an action: Choose Answer, Summarize, Improve, STAR, Bullets, Translate, Compare, or Copy.
  • Use the popup: Click the extension icon for the full interface with all controls.

Privacy

  • All personal data (profile, API keys, history) is stored locally on your device using Chrome storage.
  • Text is only sent to the AI provider you select. No intermediary servers.
  • Enable Private Mode in settings to prevent history/analytics tracking.
  • Use Export/Import in settings to backup or transfer your data.

Troubleshooting

  • "Extension context invalidated" - The extension was updated or reloaded. Refresh the webpage.
  • Toolbar not appearing - Make sure "Floating Toolbar" is enabled in Settings. Some sites with strict CSP may block it.
  • API errors - Check your API key is correct. Verify you have credits/quota with your provider.
  • No models showing - Select a provider first, then models will populate.
